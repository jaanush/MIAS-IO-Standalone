# MeTSTech Developer Kit

## Kom igång

1. Få **.deploy-env** av Johan (AZURE_PAT + COOLIFY_TOKEN)
2. Kopiera **deploy.sh** + **.deploy-env** till ditt projekt
3. Ändra APP_NAME, APP_PORT, DOMAIN i deploy.sh
4. Kör: `bash deploy.sh`
